Restrict Payment Methods

Prevent certain users from using one or multiple payment methods or Journals.

Users will only be able to register payments in allowed Journals



Instructions

1. Following the App installation, open the 'User' form for the user you wish to apply the payment methods restrictions on, and;

2. Navigate to the 'Payment Methods Restriction' tab, check the 'Restrict Payment Methods' option and specify the Journals Allowed.
