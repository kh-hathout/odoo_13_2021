Pos Multi Barcode Features:

Normally, Odoo only supports one barcode for one product. But in some case, we need to support multi barcode for one product. Pos Multi Barcode module is really helpful for you in that cases. This module not only supports to search a product with multi barcode on BackOffice, but the main purpose is the ability that allow to scan product with multi barcode on POS.

  • Compliance with Native Odoo
  • Set-up multiple barcode for product at product template or product variants on BackOffice
  • Allowing search product with multi barcode on Sale/Purchase/Inventory...
  • Allowing scan product on POS with multi barcode
