Mass Confirm Picking - Mass Confirm Delivery Order and Receipt Odoo Apps

Mass Validate Picking- Multiple Validate Delivery and Receipt Odoo Apps

Mass Confirm Delivery from Sales -Mass Confirm Receipt from Purchase Odoo Apps

Mass Confirm Delivery and Receipt Odoo Apps allow only authenticated users to confirm mass picking (Delivery/Receipt) order with a single click, Using this odoo app user don't have to manually validate/confirm all picking orders one by one. Mass Validate Picking Odoo apps are used for validating multiple picking orders in a single click whether its delivery order or receipt.These odoo apps also provide features to validate picking from sales order and purchase order too, user can be able to select multiple sales orders and confirm all delivery orders for selected sales order , same for purchase user can be able to select multiple purchase orders and confirm all receipt for selected purchase order. If selected orders are not in confirmed state then it will raise warning/validation. 

Features

Validate Mass Picking Order

User can validate multiple picking order at a time with a single click.

Validate Mass Picking Order under Sale Order

User can validate multiple picking order at a time with a single click under sale order.

Validate Mass Picking Order under Purchase Order

User can validate multiple picking order at a time with a single click under purchase order.

Warning/Validation of Mass Picking Orders

Raise warning/validation if order not in 'confirmed' state.
